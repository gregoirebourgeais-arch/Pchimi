# Démarrage enfant (sans technique)

## Windows
1. Double-clique `ouvrir_appli.bat`
2. Attends 10 à 20 secondes
3. L'app s'ouvre

## Android
1. Ouvre l'app dans Chrome
2. Menu Chrome -> `Ajouter a l'ecran d'accueil`

## Partage des donnees (sans backend)
1. Clique `Exporter mes données`
2. Envoie le fichier JSON par mail
3. Sur l'autre appareil: `Importer (fusionner)` pour ajouter sans écraser
