# Où cliquer ? (Windows, sans code)

## Important
Si tu regardes juste les fichiers sur GitHub (dans le navigateur), **tu ne peux pas lancer l'application**.
Il faut d'abord télécharger le projet sur ton PC.

## Étapes exactes

1. Sur GitHub, clique le bouton vert **Code**.
2. Clique **Download ZIP**.
3. Ouvre le ZIP téléchargé.
4. Clique **Extraire tout**.
5. Ouvre le dossier extrait.
6. Double-clique **ouvrir_appli.bat**.
7. Attends 10 à 20 secondes.
8. Le navigateur ouvre l'application sur `http://localhost:3000`.

## Pour publier sur GitHub (mise à jour)
Dans ce même dossier extrait, double-clique **publier_github.bat**.

## Si double-clic ne marche pas
- Message "Git n'est pas installé" → installe Git for Windows.
- Message "npm introuvable" → installe Node.js LTS.
- Message "python introuvable" → installe Python 3.11+.
