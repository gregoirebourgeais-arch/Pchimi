# Important: pourquoi rien ne se met sur GitHub automatiquement

Dans cette interface Codex, je **ne peux pas pousser directement sur ton dépôt GitHub**.

Le bouton en haut à droite `Créer une demande d'extraction` est l'étape obligatoire.

## Ce que tu dois faire (30 secondes)
1. Clique `Créer une demande d'extraction`.
2. Copie le patch proposé.
3. Ouvre ton dépôt GitHub.
4. Crée/édite les fichiers avec ce patch (ou applique-le via ton outil Git).
5. Commit sur `main`.

Sans cette étape, les modifications restent uniquement dans la session Codex.


Note: le projet n'utilise plus d'icones PNG binaires pour éviter l'erreur `Les fichiers binaires ne sont pas pris en charge`.
